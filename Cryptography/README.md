# Secure Chat

A simple desktop application that enables secure message exchange between users, built with Avalonia.

## Features
- Generation of encryption and signing keys for each user
- Generation of X.509 certificates using a simulated Certification Authority
- Message encryption and decryption using asymmetric cryptography
- Verification of message signatures and sender certificates
